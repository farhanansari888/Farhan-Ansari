# Farhan Ansari — Portfolio 😉

Personal portfolio website of **Farhan Ansari**, App Developer & Web Developer.

🔗 **Live:** [smartxhacker.com](https://smartxhacker.com/) · [ansarixfarhan.cloud](https://ansarixfarhan.cloud/)

## ✨ Features

- **Cinematic animations** — reassembling hero name, scroll-reveal, 3D tilt cards, magnetic buttons, custom cursor, and an animated scroll-progress bar.
- **Fully responsive** — mobile-first layout with a full-screen mobile menu.
- **SEO optimized** — Open Graph & Twitter cards, JSON-LD structured data (Person + WebSite), canonical URL, valid sitemap & robots.
- **Accessible** — semantic HTML, ARIA labels, and full `prefers-reduced-motion` support.
- **Zero build step** — pure HTML, CSS, and vanilla JS with Tailwind via CDN.

## 🛠 Tech

HTML5 · Tailwind CSS · Vanilla JavaScript · Lucide Icons
